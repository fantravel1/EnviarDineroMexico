## ‚öñÔ∏è Compliance Requirements

### FTC Guidelines (US)

- Affiliate disclosures **before CTAs**, not in footers
- Clear language explaining compensation relationship
- Disclosures visible on mobile without scrolling
- Fines up to **$50,120 per violation**

### CFPB Regulations (Financial Claims)

- No "free" claims when exchange rate spreads exist
- Speed claims require documentation
- Promotional rates must show expiration dates
- Total cost = transfer fee + exchange rate markup

### Mexico SAT Regulations (Recipient Side)

| Relationship | Tax Treatment |
|--------------|---------------|
| First-degree relatives (parents, children, spouse) | **Exempt without limit** |
| Other donations | Exempt up to ~224,000 MXN/year |
| Bank reporting threshold | 15,000 MXN/month |
